King Song Key Generator
Jonathan Vasquez <jon@xyinn.org>
Apache License 2.0

Dependencies:
Java 8

Make sure to add your java bin directory (which contains the 'java' application) to your
Environment Variables in Windows (or PATH on UNIX systems). You can find tutorials on
how to do this easily online.

Instructions:
--------------
Open a command prompt (or Terminal) window and navigate to the directory with your generator.

Type: java -jar King_Song_4.0_Key_Generator.jar

The application will begin and will ask you to enter you ISN number. After you give it
the ISN number, it will attempt to find your key.
